Anthropic ha lanzado **Claude Sonnet 4.6**, el modelo que ahora impulsa Claude Code en su modo estándar. Estas son las claves:

## Qué cambia respecto a Sonnet 4.5

- **Razonamiento de código mejorado**: mejor comprensión de contexto en proyectos grandes, menos errores en refactorizaciones complejas.
- **Menor tasa de alucinaciones**: especialmente en tareas de análisis de documentación y generación de tests.
- **Latencia reducida**: los tokens de salida llegan más rápido, lo que mejora la experiencia en sesiones de Claude Code largas.

## Cómo afecta a Claude Code

Si usas Claude Code desde el CLI, ya estás usando Sonnet 4.6 de forma automática. Puedes confirmarlo en la barra de estado inferior o ejecutando `/status`.

Para cambiar de modelo puedes usar `/model` y seleccionar Opus 4.8 si necesitas máxima capacidad, o Haiku 4.5 para tareas rápidas y baratas.

## Precios

Los precios se mantienen iguales que Sonnet 4.5 en la API. Sin cambios para usuarios de Claude Pro y Max.
